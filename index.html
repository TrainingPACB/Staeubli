Introduction

The first module gives a basic knowledge of the mechanical components of the robot arms. It explains the mandatory preventive maintenance tasks, the safety features of the robot and the machine. In addition, it covers how to control the robot via the HMI and the importance of calibration of the robot arms axes.

Intervention Level 1

The second module contains all lessons that cover the maintenance tasks. The tasks shown in this module, can be performed by any maintenance technician, without any specific training.

Intervention Level 2

This module contains all tasks which presuppose special mechanical and electrical skills, and also knowledge on how to use the service functions of the robots. These tasks can be performed by any maintenance technician, after following a dedicated training by Syntegon.
